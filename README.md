# Personal Checklist

Desktop task management application built with C# and WinForms, featuring persistent storage through a containerized MariaDB database.

## Features
- Task creation, editing, and deletion (CRUD)  
- Task prioritization and completion tracking  
- Persistent storage using MariaDB  

## Technical Highlights
- Integrated MariaDB using Docker for isolated deployment  
- Designed application logic for state management  
- Built UI with WinForms  

## Tech Stack
C#, WinForms, MariaDB, Docker
